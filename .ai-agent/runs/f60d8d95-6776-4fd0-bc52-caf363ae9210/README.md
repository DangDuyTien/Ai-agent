# Xây dựng hệ thống AI Agent tổng

Được tạo bởi Hệ thống AI Agent tổng quát MVP.

## Loại dự án

ai_tool

## Tóm tắt

công cụ AI giúp biến ý tưởng "Xây dựng hệ thống AI Agent tổng quát: người dùng nhập ý tưởng thô, agent tự phân tích loại dự án, đề xuất chức năng, kiến trúc, lộ trình, tá..." thành kế hoạch và sản phẩm có thể thực thi.

## Kiến trúc

Kiến trúc công cụ AI tập trung vào hợp đồng input/output, chuỗi prompt, model provider, đánh giá và guardrail. Chế độ sửa repo sẽ ưu tiên stack đang có: Next.js, React.

## Bước tiếp theo

Dùng PROMPTS.md với agent lập trình AI để triển khai từng tác vụ đã duyệt.
